Expérience développeur web

À ce jour j'ai appris énormément de choses concernant la programmation web, en travaillant seul ou même en groupe uniquement à distance.

HTML / CSS
J'ai commencé par étudier les bases de développement en utilisant le HTML pour structurer ma page web ainsi que le CSS pour la partie design et l'agencement d'une page web.

PHP
Par la suite j'ai exercé le PHP qui est un langage de programmation web. Ce langage permet d'ouvrir beaucoup d'accès notamment en utilisant des formulaires qui sont par la suite stockés dans une base de données. L'affectation de fichiers csv, json est extrêmement utile pour récupérer des valeurs à rajouter sur un site web.

JAVASCRIPT
Après il y a eu le Javascript, un nouveau langage front-end. Son principal atout est de supprimer le rechargement de page qui est présent sur le Php. Grâce à ce langage je me suis servi des outils des différents navigateurs qui mettent à disposition la Console, le Network, etc... Javascript permet de créer des évènements dynamiques sur les pages web.

BASE DE DONNEES / NoSQL
Les bases de données sont primordiales pour stocker les données des utilisateurs ou bien stocker des données comme par exemple une liste de tous les pays du monde avec leurs populations etc... De ce fait j'ai utilisé MySQL sur PhpMyAdmin en créant des tables avec à l'intérieur des colonnes. Grâce à cela je n'ai plus qu'à créer une class DAO avec des requêtes SQL que ce soit en SELECT ou INSERT ou UPDATE ou DELETE.
La différence avec NoSQL c'est que les données stockées ne sont pas reliées entre elles mais correspondent à des collections d'objets. NoSQL est plus flexible et permissif, mais peut aussi entrainer des problèmes de cohérence.

SYMFONY
Ce framework permet de développer et de sécuriser un site en utilisant un système de route. Toutes mes pages sont gérées par un / des contrôleur(s). Ce système permet alors de développer des fonctions différentes dans chacun de ces contrôleurs. De plus il est souvent alimenté par la communauté pour ajouter des nouvelles fonctionnalités.
